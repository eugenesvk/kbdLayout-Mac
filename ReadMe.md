## Introduction

- Unlocked <kbd>^</kbd>[^4] to enable default macOS layers 
- Extensive diacritics
- Math layer embedded ...
- Greek layer embedded behind a <kbd>üï±</kbd>, mapped to  `Greek` macOS layers (base, <kbd>‚áß</kbd>, <kbd>‚å•</kbd>, <kbd>‚áß</kbd><kbd>‚å•</kbd>), <kbd>üï±</kbd>`Greek` (mapped to <kbd>^</kbd><kbd>g</kbd>), and (in <kbd>üï±</kbd>`Greek`):
        * <kbd>üï±</kbd>`greek-tonos` (mapped to <kbd>;</kbd>)
        * <kbd>üï±</kbd>`diaeresis-greek` (mapped to <kbd>‚áß</kbd><kbd>;</kbd>)
        * <kbd>üï±</kbd>`greek-dialytika-tonos` (mapped to <kbd>‚áß</kbd><kbd>w</kbd> / <kbd>‚å•</kbd><kbd>;</kbd>)
[^4]: <kbd>‚áß</kbd> is <kbd>Shift</kbd>, <kbd>^</kbd> is <kbd>Control</kbd>, <kbd>‚å•</kbd> is <kbd>Alt</kbd>, <kbd>üï±</kbd> is dead key


## Installation

- Copy `TypES Layout.bundle` to `~/Library/Keyboard Layouts`
- Open `System Preferences` ‚Üí `Keyboard` ‚Üí `Input Sources` ‚Üí `+` to add:
    + `English ‚Äî TypES`
    + `Russian ‚Äî TypES`
    + `English ‚Äî Math` (though it is also embedded in the English/Russian layouts)
- (_optionally_) Delete the [default layout](https://apple.stackexchange.com/questions/44921/how-to-remove-or-disable-a-default-keyboard-layout#60521). For macOS `10.9` and later:
    + Change the current input source to your custom keyboard layout added above
    + Backup then open `~/Library/Preferences/com.apple.HIToolbox.plist` (you can convert the plist to XML with `plutil -convert xml1`)
    + Remove the input source you want to disable from the `AppleEnabledInputSources` dictionary
    + Remove the `AppleDefaultAsciiInputSource` key if it exists
    + Restart

## Usage


To allow entering all the extra symbols these keyboard layouts rely heavily on <kbd>üï±</kbd>s (or dead keys), which are keys that don't print anything by themselves, but instead enter into a new keyboard layer with extra symbols. For example[^3] <kbd>‚å•</kbd><kbd>c</kbd> is a <kbd>üï±</kbd> dead key `Copyright`, pressing it enables to enter the following 3 symbols, each with just a single key:

  1. `¬©` with <kbd>c</kbd>
  2. `¬Æ` with <kbd>r</kbd>
  3. `‚Ñ¢` with <kbd>t</kbd>

[^3]: Most examples are using the English layout, though most of the keybinds are identical in the Russian layout as well (<kbd>‚å•</kbd><kbd>o</kbd>‚â°<kbd>‚å•</kbd><kbd>—â</kbd>)

How to find how to insert a symbol?

- Open the [SymbolsAll-En](./helper/SymbolsAll-En.md) or [SymbolsAll-Ru](./helper/SymbolsAll-Ru.md) files for the English/Russian layouts, find a symbol and look at its row/column re. which modifier+key combo produces it
- (certain, but tedious alternative): find a symbol in the keyboard layout file (e.g. `English ‚Äî TypES.keylayout`) and note the `key code` and `keyMap index` tha produce it, lookup the values in Ukelele

### Additional tips

#### Insert English <kbd>‚áß</kbd><kbd>1</kbd>‚Äì<kbd>0</kbd> symbols in the Russian layout

Russian layout has somehwat different <kbd>‚áß</kbd><kbd>1</kbd>‚Äì<kbd>0</kbd> symbols:<br>
<kbd>2</kbd><kbd>3</kbd><kbd>4</kbd><kbd>6</kbd><kbd>7</kbd><br>

  1. Use <kbd>^</kbd><kbd>‚å•</kbd> that copies the <kbd>‚å•</kbd> layer of the `Russian - PC` layout  
  2. Use `Karabiner-Elements` to remap <kbd>right‚áß</kbd> to insert English characters in the Russian layout (unfortunately, `Ukelele` can't differentiate between left and right keys):
    + copy this [config file](./helper/ru-RShift=en-LShift.json) to `~/.config/karabiner/assets/complex_modifications`
